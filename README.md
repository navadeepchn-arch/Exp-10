# Experiment 10

## Title  
Todo Application with CRUD Operations using React

---

## Description  
This project implements a simple Todo application using React.js to demonstrate CRUD (Create, Read, Update, Delete) operations using frontend state management. The application uses React hooks to manage the list of tasks, handle user input, and dynamically update the UI based on user actions.

The experiment focuses on understanding how data flows in a React application and how state can be used to build interactive applications without a backend database.

---

## Learning Outcomes  
- Understand CRUD operations in frontend applications  
- Implement state management using React hooks (useState)  
- Use arrays and objects to manage dynamic data  
- Perform create, update, and delete operations on UI data  
- Apply conditional rendering in React  
- Build interactive and responsive UI components  

---

## Tech Stack  
- React 18+  
- JavaScript (ES6+)  
- HTML5  
- CSS3  
- Node.js (React environment)

---

## Features  
- Add new todo tasks  
- Display list of tasks  
- Mark tasks as completed/uncompleted  
- Delete tasks from list  
- Dynamic UI updates using React state  
- Clean and responsive design  

---

## How to Run Locally  

### Step 1: Install dependencies  
npm install  

### Step 2: Start development server  
npm start  

### Step 3: Open in browser  
http://localhost:3000
